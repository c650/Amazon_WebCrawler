README.txt for asinReview.py
This is based on the original crawler created by ScrapeHero.
A crawler used to scrape Amazon by a product's asin (product code) and returns the product's
rating and total number of reviews in a formatted excel sheet. Knowledge of lxml, xlxswriter and
basic python is required.

Input should be a text file named asins.txt in this format:
B00005O3VC
B01AYAKEGK
No spaces included, one asin after the other.
